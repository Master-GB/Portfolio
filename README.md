# Portfolio — GihanOS

Concept: **GihanOS** — A personal developer operating system portfolio.

## Features

- 🖥️ **Mini OS desktop** — draggable windows, taskbar, start menu
- 🤖 **3D robot assistant** — interactive Byte bot powered by Three.js
- ✨ **Animations** — boot sequence, window transitions, skill bars (Framer Motion)
- 📂 **Portfolio apps** — About, Projects, Skills, Resume, Education, Contact, Terminal, Settings
- 🎨 **Customizable** — wallpapers, reduce motion, toggle 3D robot

## Getting Started

```bash
cd gihan-os
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Customize Your Portfolio

Edit **`gihan-os/src/data/portfolio.ts`** with your:

- Name, bio, email, social links
- Projects and tech stack
- Skills and experience
- Education details

## Tech Stack

- Next.js 16 · React 19 · TypeScript
- Tailwind CSS 4
- Framer Motion · React Three Fiber · Three.js
- Lucide Icons

## Deploy

Deploy to [Vercel](https://vercel.com) or any static host that supports Next.js.
